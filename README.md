# 🏛️ بيت المال — Bayt Al-Mal

تطبيق إدارة المصروفات الشخصية باللغة العربية.

---

## 🚀 خطوات الرفع على Vercel (مجاناً)

### الخطوة 1 — رفع المشروع على GitHub
1. اذهب إلى [github.com](https://github.com) وسجل دخولك (أو أنشئ حساباً مجاناً)
2. اضغط **New Repository**
3. اسم المستودع: `bayt-al-mal`
4. اضغط **Create Repository**
5. ارفع جميع ملفات المشروع

### الخطوة 2 — ربط Vercel
1. اذهب إلى [vercel.com](https://vercel.com)
2. سجل دخولك بحساب GitHub
3. اضغط **New Project**
4. اختر مستودع `bayt-al-mal`
5. اضغط **Deploy** — وخلاص! ✅

سيعطيك رابطاً مثل: `bayt-al-mal.vercel.app`

---

## 💻 تشغيل محلياً

```bash
npm install
npm run dev
```

ثم افتح: http://localhost:3000

## 🗂️ هيكل المشروع

```
src/
├── app/
│   ├── layout.tsx      # الـ layout الأساسي
│   ├── page.tsx        # الصفحة الرئيسية والـ navigation
│   └── globals.css     # الأنماط العامة
├── components/
│   ├── ui.tsx          # مكونات مشتركة (StatCard, Badge, ProgressBar)
│   ├── Dashboard.tsx   # لوحة التحكم
│   ├── AddExpense.tsx  # إضافة مصروف
│   ├── ExpensesList.tsx # قائمة المصروفات
│   ├── Analytics.tsx   # التحليلات
│   └── Pages.tsx       # باقي الصفحات
└── lib/
    └── data.ts         # البيانات والثوابت والأنواع
```
